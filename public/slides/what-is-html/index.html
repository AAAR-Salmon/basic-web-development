HyperText Markup Language

ここが重要

文書の構造や意味をタグによって示すことができる

 
<!DOCTYPE html>
<html lang="ja">
  <head>
    <meta charset="UTF-8">
    <title>hello</title>
  </head>
  <body>
    <h1>こんにちは</h1>
    <p>これがHTMLです。</p>
  </body>
</html>
